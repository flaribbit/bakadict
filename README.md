# dict

离线的命令行日语词典

![demo](https://user-images.githubusercontent.com/24885181/212700770-90a154e2-9bef-4822-b1ea-1ad2d25edcf1.png)

## 使用方法
- `dict 割合`
- `dict わりあい`  
  支持使用汉字或假名查询
- `dict wariai`  
  来回切换输入法很蛋疼，因此本程序还支持使用罗马音查询！
- `dict wariai -`  
  习惯上完全匹配的结果在最前面，终端输出后默认视图在底部，可以末尾添加 `-` 来反向输出查询结果。
